这个仓库的内容，是对通山赣语的介绍/输入法工具获取页面，
概述
  此仓库中，有一个简单的html输入工具，可让你在不使用任何输入法的情况下，输入正确的赣语文本，输入方式一部分参考了越南输入法。
声调系统
  通山赣语有六个声调，前四个声调和中文一样，第五声是第一声的降调，第六声是第三声的降调。在输入时，有一些按键被加载音节结尾，这些音节是声调符号。列表如下。
1.	j,
2.	.r
3.	.d
4.	.f
5.	T
6.	L

拼写方式和中文一样，辅音，元音/元音字母组合，声调。赣语中有一些特殊的拼法，则按拉丁字符组合的方式拼出。例如io，实际读音是短i加o。字母读法和中文拼音有些类似，你可以在仓库中下载字母表朗读录音。
